# 7차시 프롬프트 모음

## 1번 프롬프트
```text
백테스트 시스템의 핵심 구성 요소를 설명해줘. 
데이터 로딩, 전략 실행, 성과 분석, 
결과 시각화의 각 단계를 자세히 설명해줘.
```

## 2번 프롬프트

```text
실제 거래와의 차이점을 분석해줘.
```

```text
pandas를 사용해서 효율적으로 작성해줘.
```
