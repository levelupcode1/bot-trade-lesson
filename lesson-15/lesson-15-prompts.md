# 15차시: 자동매매 시스템 배포 및 운영

## 1번 프롬프트

```text
[배포 목표]
- 24시간 안정적인 운영, 빠른 배포 및 롤백
- 비용 효율적인 운영
[인프라 요구사항]
1. 컴퓨팅 리소스
   - CPU: 2-4 코어, 메모리: 4-8GB RAM
   - 스토리지: 50-100GB SSD
   - 네트워크: 고속 인터넷 연결
2. 운영체제 및 런타임
   - Ubuntu 20.04 LTS, Docker 지원
   - Python 3.9+, Node.js 16+ (웹 UI용)
3. 데이터베이스
   - PostgreSQL 13+
   - 백업 및 복구 시스템
[보안 요구사항]
- API 키 보안 관리, 방화벽 설정
- 접근 권한 관리, 로그 모니터링
[비용 요구사항]
- 월 예산: 50-100만원, 비용 모니터링
이 요구사항을 만족하는 클라우드 플랫폼을 추천해줘.
```
