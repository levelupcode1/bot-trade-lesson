## 시작하기

### 1단계: 시스템 설치

```bash

cd lesson-14

# 의존성 설치
pip install -r requirements.txt
```

## 사용자 유형 선택

시스템에 처음 로그인할 때, 자신의 투자 경험과 리스크 성향에 맞는 유형을 선택하세요.

### 🌱 초보자 (권장: 투자 경험 1년 미만)

**특징:**
- 최대한 안전한 거래 환경
- 단순하고 직관적인 인터페이스
- 모든 항목에 설명 제공
- 엄격한 리스크 관리

**제한사항:**
- 일일 거래 3회로 제한
- 포지션 크기 15%로 제한
- BTC, ETH만 거래 가능
- 손절 라인 -3% 고정

**추천 대상:**
- 암호화폐 투자 초보자
- 안정적인 수익을 원하는 분
- 적은 금액으로 시작하는 분

### 📊 중급자 (권장: 투자 경험 1-3년)

**특징:**
- 균형잡힌 기능 제공
- 전략 파라미터 조정 가능

- 전략을 커스터마이징하고 싶은 분

## 웹 인터페이스 사용법

### 초보자 대시보드

초보자 대시보드는 가장 필수적인 정보만 표시합니다.

#### 메인 화면 구성

```
┌─────────────────────────────────┐
│ 💰 내 계좌                       │
│   총 자산: 3,250,000원           │
│   오늘 수익: +25,000원 (+0.77%) │
│   현금: 1,625,000원 (50%)       │
│   🟢 시스템 작동 중              │
├─────────────────────────────────┤
│ 🪙 보유 중인 코인               │
│   (현재 보유 내역)              │
├─────────────────────────────────┤
│ ⚡ 최근 거래 (오늘)             │
│   (오늘의 거래 내역)            │
├─────────────────────────────────┤
│ ⚙️ 간단 설정                    │
│   전략: 변동성 돌파 (안전)       │
│   손절: -3% (변경 불가)          │
│   익절: +5%                      │
└─────────────────────────────────┘
```

#### 주요 기능

**1. 계좌 현황 확인**
- 총 자산과 현금 비율 확인
- 오늘의 수익/손실 확인
- 시스템 상태 확인

**2. 보유 포지션 확인**
- 현재 보유 중인 코인
- 각 코인의 수익률
- 매수 가격과 현재 가격

**3. 거래 내역 확인**
- 오늘 실행된 거래
- 거래 횟수 (3회 제한)
- 매수/매도 시간과 가격

**4. 설정 변경**
- 익절 목표 조정
- 알림 설정
- 전략 확인

## 전략 선택 및 설정

### 초보자용 전략

#### 1. 변동성 돌파 전략 (보수적)

**설명:**
가격이 전날 변동폭의 30%만큼 상승하면 매수하는 안전한 전략입니다.

**설정:**
- K값: 0.3 (고정)
- 손절: -3%
- 익절: +5%

**사용 방법:**
1. 대시보드에서 "전략" 메뉴 클릭
2. "변동성 돌파 (보수적)" 선택
3. 익절 목표만 조정 가능 (5-10%)
4. "적용" 버튼 클릭

---

### 중급자용 전략

#### 1. 변동성 돌파 전략

**설정 가능 항목:**
- K값: 0.3 ~ 0.7
- 손절: -5% ~ -10%
- 익절: +5% ~ +15%

#### 2. 이동평균 교차 전략

**설정:**
- 단기 이동평균: 5 ~ 20일
- 장기 이동평균: 20 ~ 60일
- 손절/익절 설정

#### 3. RSI 전략

**설정:**
- RSI 기간: 7 ~ 21일
- 과매수 기준: 60 ~ 80
- 과매도 기준: 20 ~ 40

#### 4. 포트폴리오 전략

**설정:**
- 코인 선택 (최대 5개)
- 각 코인 비중 설정
- 리밸런싱 주기

## 프로필 업그레이드

### 업그레이드 조건

## 문제 해결

### 일반적인 문제

**Q: 로그인이 안 돼요**
A: 사용자 ID를 정확히 입력했는지 확인하세요. 프로필이 없으면 자동으로 생성됩니다.

**Q: 거래가 실행되지 않아요**
A: 
1. 일일 거래 한도 확인
2. 잔고 확인
3. 허용된 코인인지 확인

**Q: 손절 라인을 변경하고 싶어요 (초보자)**
A: 초보자는 안전을 위해 -3%로 고정되어 있습니다. 중급자로 업그레이드 후 변경 가능합니다.

**Q: 더 많은 코인을 거래하고 싶어요**
A: 프로필을 업그레이드하세요. 중급자는 TOP 30, 고급자는 모든 코인이 가능합니다.

### 에러 메시지

**"포지션 크기가 제한을 초과합니다"**
- 해결: 투자 금액을 줄이세요
- 원인: 계좌 대비 너무 큰 포지션

**"거래 허용 목록에 없습니다"**
- 해결: 다른 코인 선택 또는 프로필 업그레이드
- 원인: 현재 프로필에서 허용하지 않는 코인

**"일일 거래 한도 초과"**
- 해결: 내일까지 기다리거나 프로필 업그레이드
- 원인: 오늘의 거래 횟수 소진
